## Deezepy
``` python
>>> import deezepy
>>> client = deezepy.Client()
>>> client.get_track(760429392).title
'Come Together'
```
**Deezepy** is an elegant, intuitive and easy-to-use Python wrapper for Deezer API.

### Installation

```pip install deezepy```

### Requirements
- Python 3.6 or higher
